## Let’s talk about Present Tense!

All verbs change depending on if you’re talking about je, tu, and so on!

| subject | verb (parler) |
| ------- | ------------- |
| je | parl**e** |
| tu | parl**es** |
| il / elle | parl**e** |
| nous | parl**ons** |
| vous | parl**ez** |
| ils / elles | parl**ent** |
